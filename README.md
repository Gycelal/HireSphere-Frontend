# HireSphere Frontend

Frontend for **HireSphere**, built with **React + Vite**.  
Handles UI for Candidates, Recruiters, and Admins.

## Features

- Role-based authentication (Candidate, Recruiter, Admin)
- Separate login pages for each user type
- Basic dashboards for all users
- Theme toggle (light/dark) with Redux state
- Responsive UI with Tailwind CSS

## Getting Started

### Prerequisites

- Node.js v18+  
- npm or yarn  

### Install Dependencies

```bash
npm install
# or
yarn install
